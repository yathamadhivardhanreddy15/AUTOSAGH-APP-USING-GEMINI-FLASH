# AutoSage Gemini App

A simple Generative AI web app that analyzes images using Google's Gemini Flash model.

## Features
- Upload an image
- Get AI-generated description
- Built with Flask
- Secure API key using .env

## Setup

1. Install dependencies:
pip install -r requirements.txt

2. Add your Google API key to `.env`

3. Run the app:
python app.py

4. Open http://127.0.0.1:5000
